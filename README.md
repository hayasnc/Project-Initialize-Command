# Project Initialize command
Usage
```
git clone "https://github.com/hayasnc/Project-Initialize-Command.git"
cd to the folder
pip install -r requirements.txt
vi .env
Then open the .env file and store your username, password, and desired file destination. Use format in the sample.env file
source ~/.custom_commands.sh

To run the script type in 'create_project <name of your folder>'

```
